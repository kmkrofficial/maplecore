# Scripts

Operational utilities for training, data generation, and assets.

## Training
- **`train_generalist.py`**: The main training loop for MAPLE models. Includes hardware profiling.
- **`train_needle_finetune.py`**: Script for fine-tuning the model on specific needle retrieval tasks.

## Data Preparation
- **`generate_oracle.py`**: Downloads and pre-processes the NarrativeQA dataset.
- **`generate_oracle_hotpot.py`**: Downloads and pre-processes the HotpotQA dataset.

## Reporting
- **`generate_paper_assets.py`**: Generates high-quality plots and figures for the research paper.
- **`verify_linear.py`**: Quick sanity check script for model inference.
